# iRent v1.0 - Rental Property Management System

Production-ready rental management web application with React 19, Next.js 16, and Material Design 3.

## 🚀 Quick Start

```bash
npm install && npm run dev
# Visit http://localhost:3000
# Login: any@email.com / any password
```

## ✨ Features

- Authentication with sign-in/registration
- 5-tab responsive dashboard (Rooms, Reports, Sales, Updates, Chat)
- 3-step billing wizard with meter simulation
- Interactive payment tracking with status updates
- Sales analytics with ROI metrics
- Subscription tier management
- Split-pane messaging system
- Material Design 3 interface with smooth animations
- Full TypeScript type safety
- Mobile-optimized responsive design
